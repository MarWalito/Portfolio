Bienvenue sur mon projet !

Présentation

Langages utilisés : HTML - CSS - JS - PHP - SQL

Durée et date : 2 Semaines | Juillet 2021

J'ai créé ce site pour un ami à moi qui possède un serveur de jeu. C'est un site vitrine, il y a donc aucune connexion à une base de données derrière.

Détails
Pour commencer, j'ai donc demander à mon ami ce qu'il souhaitait avoir sur son site et par la suite j'ai donc établis différentes maquettes. Le site comporte en tous 7 pages, il y a l'accueil, le règlement, les différents catalogues et la boutique pour soutenir le serveur. Ce site a pour objectif de servir de vitrine au serveur afin de présenter le serveur aux nouveaux joueurs. Il permet aussi de posséder les choses utiles pour les joueurs réguliers.
