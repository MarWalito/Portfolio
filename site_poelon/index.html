Bienvenue sur mon projet !

Présentation

Langages utilisés : HTML - CSS - JS - PHP - SQL

Durée et date : 6 Semaines partie front-end - 6 Semaines partie back-end | Janvier 2021 - Avril 2021

Collaboration : Benjamin pour la partie front-end puis tout seul pour le back-end

Le site le petit Poêlon est un site d'e-commerce sur lequel nous pouvons rencontrer différents types nécessaires pour faire une raclette.

Détails
Au début nous avons réaliser le front-end en binome, pour cela nous avons donc voulu faire un site sur la vente de produit de raclette. Nous avons donc créer différentes pages avec des produits liés à la raclette. Par la suite j'ai donc fait le back-end du site, avec notamment un affichage dynamique des produits, un panier propre à chaque utilisateur ainsi qu'une partie admin pouvant, ajouter, supprimer et modifier les produits présents dans la bdd.

Compétences développées
- Développer la présence en ligne de l’organisation
- Travailler en mode projet
- Gérer le patrimoine informatique
